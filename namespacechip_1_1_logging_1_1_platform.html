CHIP log output functions.

Log, to the platform-specified mechanism, the specified log message, msg, for the specified module, module, in the provided category, category.

CHIP log output function.

Parameters
[in]moduleThe name of the log module.
[in]categoryA LogCategory enumeration indicating the category of the log message. The category may be filtered in or out if CHIP_LOG_FILTERING was asserted.
[in]msgA pointer to a NULL-terminated C string with C Standard Library-style format specifiers containing the log message to be formatted and logged.
[in]vA variadic argument list whose elements should correspond to the format specifiers in msg.
